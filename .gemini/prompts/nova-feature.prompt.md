# Prompt: nova-feature

**Uso:** Peça "/nova-feature <descrição>"

**Ação:**
1. Acione a lógica de **Specifier**.
2. Crie o arquivo de feature correspondente.
3. Atualize o índice de features.

**Handoff Final:**
Gere um Handoff para o comando `/proxima-feature`, indicando o nome da branch e as tarefas prioritárias da nova feature.
